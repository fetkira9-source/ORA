# ORA - Warehouse Management System

## Objetivo

ORA es un sistema de gestión de almacén (WMS) desarrollado para controlar todos los procesos logísticos de una empresa.

## Módulos

- Recepción
- Ubicaciones
- Picking
- Conferencia
- Expedición
- Inventario
- Auditoría

## Tecnologías

- Java 21
- Spring Boot
- Kotlin
- SQL Server
- React
